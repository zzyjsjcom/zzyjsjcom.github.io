1. Introduction

随着人工智能技术、高性能计算应用越来越广泛，与之关系密切的异构计算平台也越来越普及。 目前比较流行的两大计算平台分别是 OpenCL 和 CUDA。 如果想学习这方面的技术，购买 GPU 是一条常见的路径。本文会提供一些相对低成本一点的方法。 目前几乎稍微主流一点的 intel 处理器，都会有集成显卡，因此我们会介绍OpenCL在intel集成显卡上环境搭建的方法。 Nvidia 的显卡一般都很贵，但是很多平台提供了免费平台可以使用 Nvidia 的显卡，我们会以百度的飞桨平台为例做介绍。

2. OpenCL setup on Intel Integrated GPU

本节内容可以对照 《计算机系统开发与优化实战》第7章的内容。

准备工作

如果你的计算机有独立显卡，比如 AMD 显卡或者 Nvidia 显卡，那么可以直接安装独立显卡的驱动来学习 OpenCL 知识。 AMD 和 Nvidia 独立显卡安装驱动，网上有很多教程，大家可以跳过本节内容。 否则，如果你的cpu是intel的，那么可以通过下面命令查看你的cpu是否包含集成显卡。如果不包含，那么就没办法搭建 OpenCL 实验环境了。 运行如下命令，如果你的输出跟下面这个输出很类似，那么说明你的intel cpu 包含了集成显卡，可以继续。

 
$ lspci | grep -i vga
00:02.0 VGA compatible controller: Intel Corporation Skylake GT2 [HD Graphics 520] (rev 07)

安装一些通用依赖包

 
# install OpenCL common package
sudo apt install ocl-icd-libopencl1
sudo apt install opencl-headers
sudo apt install clinfo
sudo apt install ocl-icd-opencl-dev
sudo apt install intel-opencl-icd

添加 package repository

Note: 这里要特别注意，以下的命令是针对 ubuntu-20.04 的，如果你是其他版本的 ubuntu 系统，那么需要把 focal 之类的字样修改成对应发行版的代号。

 
sudo apt-get install -y gpg-agent wget
wget -qO - https://repositories.intel.com/graphics/intel-graphics.key |
  sudo apt-key add -
sudo apt-add-repository \
  'deb [arch=amd64] https://repositories.intel.com/graphics/ubuntu focal main'

安装 GPU run-time 软件包

 
sudo apt-get update
sudo apt-get install \
  intel-opencl-icd \
  intel-level-zero-gpu level-zero \
  intel-media-va-driver-non-free libmfx1

安装开发者软件包

 
sudo apt-get install \
  libigc-dev \
  intel-igc-cm \
  libigdfcl-dev \
  libigfxcmrt-dev \
  level-zero-dev

其它配置

 
stat -c "%G" /dev/dri/render*
groups ${USER}
sudo gpasswd -a ${USER} render
newgrp render

3. CUDA free platform Introduction

本节内容可以对照 《计算机系统开发与优化实战》这本书第6.2节的内容。

如果你有 nvidia 的独立显卡，那么可以跳过本节内容。网上有很多 nvidia GPU 驱动的安装方法可以参考。 本节内容介绍百度飞桨平台提供的免费 nvidia GPU 开发环境（当然它是有很多限制的）。

首先打开网站: https://aistudio.baidu.com/aistudio/index ，注册一个账号，然后登录。 首先点击最上面菜单栏的 “项目”，然后点击“创建项目”。按照自己的喜好，进行一系列的操作。 项目创建完成之后，从“我的项目”里面可以找到您刚才创建好的项目。 点进去，找到“启动环境”，然后选择一个带 GPU 卡的配置。 这样就进入到了你的项目。这个时候，你会看到一个叫“main.ipynb” 的python程序文件，点击这个程序右边的叉，把这个程序关闭。 然后点击“终端”，这样你就进入命令行了，可以通过git命令从网站上下载你的cuda程序，就可以编译运行了。
