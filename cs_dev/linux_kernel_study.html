1. BiscuitOS Introduction

通过如下命令拉取代码，并编译

 
https://gitee.com/zzyjsjcom/biscuitos_prebuild.git
sudo apt-get install gcc-aarch64-linux-gnu
make prepare image

运行如下命令，进入BiscuitOS环境

 
make run

BiscuitOS会在 /mnt 下自动挂在上你的 home 目录，假设你的代码放到了 ~ 目录，那么切换到如下目录， 并执行su切换到root用户，执行make运行

 
cd /mnt/biscuitos_prebuild/test_ko/
su
# make
