2. Get and Run RISCV64 Docker

使用以下两条命令，你就可以在x86 ubuntu的环境中搭建起一个riscv64的学习环境了。

 
sudo docker pull ft2team/riscv64_u2004
sudo docker run --rm -v /mnt:~ -it ft2team/riscv64_u2004
